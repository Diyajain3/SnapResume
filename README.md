## ✨ Overview

**SnapResume** is a full-stack Resume Builder application designed to help users create clean, professional resumes with ease.

The project is currently focused on building a smooth and responsive frontend experience, with backend features planned next.

---

## 🚧 Current Status

🔨 **Frontend in development**
🧠 Backend architecture planned
🚀 Full-stack version coming soon

## 🎯 Features

### ✅ Implemented (Frontend)

* Responsive and modern UI
* Form-based resume input
* Component-based architecture (React)
* Clean layout for resume preview

### 🔜 Coming Soon

* 🔐 User Authentication
* 💾 Save resumes (Database)
* 📄 Download as PDF
* 🎨 Multiple templates
* ☁️ Cloud storage

## ⚙️ Getting Started

```bash id="l7q9mz"

git clone https://github.com/Diyajain3/SnapResume.git

# Move into project
cd SnapResume

# Install dependencies
npm install

# Run project
npm start
```
